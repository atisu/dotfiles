name: obisidian-notes
description: A skill for working with an Obsidian vault through the qmd MCP. Use it when a user wants to find notes, surface linked ideas, summarize note content, trace relationships between concepts, or turn scattered vault material into clear answers and insights.
---

Use the qmd MCP to search and read an Obsidian vault by note content, metadata, tags, and links. This skill is useful when a user asks to retrieve specific notes, connect related ideas, summarize a cluster of notes, explore how concepts relate across the vault, or synthesize existing notes into a more helpful explanation.
