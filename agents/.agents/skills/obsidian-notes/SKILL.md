---
name: obsidian-notes
description: A skill for working with an Obsidian vault through the qmd MCP. Use it when a user wants to find notes, surface linked ideas, summarize note content, trace relationships between concepts, or turn scattered vault material into clear answers and insights.
---

Use the qmd MCP to search and read an Obsidian vault by note content, metadata, tags, and links. This skill is useful when a user asks to retrieve specific notes, connect related ideas, summarize a cluster of notes, explore how concepts relate across the vault, or synthesize existing notes into a more helpful explanation.

Use the qmd MCP to refresh embeddings and add new notes to the vault when the user wants to update the knowledge base with new information or ensure that recent changes are reflected in the agent's understanding. This is especially important when the user has added new notes, made significant edits, or wants to ensure that the agent has the most up-to-date information from the vault for accurate responses. Use the 'update' functionality first to add new notes and then the 'refresh' functionality to update the embeddings so that the agent can effectively retrieve and utilize the new information in future interactions.
